- Uma das vantagens de um boa organização de código é a fácilitade de integrar de novas pessoas para dar manutenção ao sistema
 
 Algumas caracteriscas de um software de qualidade é:
 - Fácilidade para usar;
 - Funcionamento correto;
 - Fácilidade de dar manutenção;
 - Mantém integridade dos dados;

 As linguagens tem alguns paradigma que são:
- Programação Procedural.
- Programação Orientada ao Objeto.
- Computação Paralela.
- Paradigma de Lógica de Programação.
- Programação Funcional.
- Declarativo: Onde o programador apenas declara propriedades do resultado desejado, mas não como calculá-lo;
- Imperativo: O programador instrui a máquina sobre como alterar seu estado

UML - Unified Modeling Language:
Linguagem de modelagem de desenvolvimento geral, de uso geral, no campo da engenharia de software, cujo objetivo é fornecer uma maneira padrão de visualizar o projeto de um sistema

Uso incorreto do UML é você querer projetar todas as partes do sistema com ele, ou assumir que novatos consegue projetar com UML
